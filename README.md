# OrdenacaoBubble

Projeto Java simples que ordena 10 números usando algoritmo Bubble Sort.

## Estrutura do projeto

- `pom.xml` - arquivo de configuração Maven
- `src/main/java/com/example/ordenacao/OrdenacaoBubble.java` - código-fonte Java

## Como executar

```bash
cd ~/projetos
mvn compile
mvn exec:java
```

Ou diretamente com classes compiladas:

```bash
java -cp target/classes com.example.ordenacao.OrdenacaoBubble
```
